Capítulo 2 Séries Temporais no Contexto de Regressão

Neste capítulo abordamos regressão no contexto de séries temporais. Começamos definindo o que é uma série temporal e introduzimos algumas propriedades teóricas.

2.1 Introdução

Uma série temporal é qualquer conjunto de observações ordenadas no tempo. Alguns exemplos são citados abaixo:

  1. Estimativas trimestrais do Produto Interno Bruto (PIB);

  2. Valores diários da temperatura em Campo Bom;

  3. Índices diários da bolsa de valores de São Paulo;

  4. Quantidade anual de chuva na cidade do Recife;

  5. Um registro de marés no porto de Santos.

Nos exemplos de a) a d) temos séries temporais discretas, enquanto que e) é um exemplo de série contínua. Podemos obter uma série temporal discreta a partir da amostragem de uma série temporal contínua considerando intervalos de tempos iguais, \(\Delta t\). Assim para analisar a série e) será necessário amostrá-la, convertendo-a e observando-a no intervalo de tempo \([0,T]\), supondo uma série discreta com \(N\) pontos, em que \(N = \Delta t/T\) (T horas). Existem dois enfoques utilizados na análise de séries temporais. Em ambos, o objetivo é construir modelos para estas séries. No primeiro enfoque, a análise é feita no domínio temporal e os modelos propostos são modelos paramétricos (com um número finito de parâmetros). No segundo, a análise é conduzida no domínio de frequências e os modelos propostos são modelos não-paramétricos. Dentre os modelos paramétricos temos, por exemplo, os modelos ARIMA, que serão estudados neste curso nos próximos capítulos. No domínio de frequências temos a análise espectral, que tem inúmeras aplicações em ciências físicas e engenharia, principalmente na engenharia elétrica, e que consiste em decompor a série dada em componentes de frequências e onde a existência do espectro é a característica fundamental. Este tipo de análise não será estudado nestas notas de aulas, para detalhes o aluno deve consultar Jenkins e Watts (1968), Koopmans (1974), Morettin (1979), Marple (1987) e Kay (1988).

2.2 Exemplos de Séries Temporais

Exemplo 1.1 Vamos supor que desejamos medir a temperatura máxima do ar, de um local, durante 24 horas, poderíamos obter um gráfico semelhante a figura abaixo: 
## Warning: package 'plotly' was built under R version 3.6.3
#library(readxl)
#library(tidyr)
#library(dplyr)
#library(ggplot2)
# Dados baixados em 23/06/2020 de
# http://sisdagro.inmet.gov.br/sisdagro/app/monitoramento/bhs

TempPortoAlegreA <- read_excel("Dados/TempPortoAlegreA.xls")
TempManaus <- read_excel("Dados/TempManaus.xls")
DATA=seq(as.Date("2011-01-02"), as.Date("2020-06-23"), by="days")
Temp=data.frame(DATA=DATA,POA=TempPortoAlegreA$TEMPERATURA,
              MANAUS=TempManaus$TEMPERATURA)

df <- Temp[(length(Temp$DATA)-365):length(Temp$DATA),] %>%
  select(DATA,POA,MANAUS) %>%
  gather(key = "Temperatura", value = "valor", -DATA)

ggplot(df, aes(x = DATA, y = valor)) + 
  geom_line(aes(color = Temperatura), size = 0.5) +
  scale_color_manual(values = c("#00AFBB", "#E7B800")) +
theme_minimal()+labs( x="", y="")+
  annotate(geom="text", y=27,x=as.Date("2020-07-30"), col="#00AFBB", 
           label="Z_1", parse=T,size =6)+
  annotate(geom="text", y=20,x=as.Date("2020-07-30"), col="#E7B800", 
           label="Z_2", parse=T,size =6)

Figura 2.1: Temperaturas médias diárias nas cidades de Manaus (MA) e Porto Alegre (POA)

Cada curva do gráfico é chamada de trajetória ou série temporal ou função amostral. No gráfico acima \(Z_{(j)}(t)\) é o valor da temperatura no instante \(t\), para a \(j\)-ésima trajetória (\(j\)-ésimo ponto de observação). Para cada \(t\) fixo, teremos os valores de uma variável aleatória \(Z(t)\) que terá certa distribuição de probabilidade. Na realidade o que chamamos de série temporal, é uma parte de uma trajetória, dentre muitas que poderiam ter sido observadas. O parâmetro \(t\) pode ser função de algum outro parâmetro físico como por exemplo: espaço e volume.
